# Weather-app
The Weather App is a simple web application built with HTML, CSS, and JavaScript. It uses the OpenWeather API to fetch real-time weather details such as temperature, humidity, and wind speed for any city entered by the user.
# Weather App
Simple weather app using OpenWeather API.  
**Tech:** HTML • CSS • JavaScript

## Setup
1. Get OpenWeather API key.
2. Create a `.env` file or use server to keep the key secret.
3. Open `index.html` in a browser.

## Features
- Search by city
- Shows temperature, humidity, wind, and condition
